# Local Run

```
git clone https://github.com/nekozu/NekozuUserbotPython

cd NekozuUserbotPython

cd NekozuUserbotPy

pip3 install -r requirements.txt

and satisfy the values in .env

python3 -m NekozuUserbotPy
```
Done!
